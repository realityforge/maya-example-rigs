# Maya Example Rigs

A collection of freely available rigs that I have collected to learn from. The rigs, models and textures are all the works of other individuals with minor touchups from me.

## Tiger Anthro rig

Features:
* IK/FK arms
* Dynamic/FK tail
* Breathing Control
* Layered controls
